# Flappy Bird

Flappy Bird ist ein Spiel, das eingestellt wurde, weil seine Schöpfer fanden, dass es zu süchtig macht. 
In Flappy Bird muss ein Vogel zwischen zwei Rohren fliegen, ohne sie zu berühren. 
Ständig tauchen neue Röhrenpaare auf. 
Der Vogel fällt immer wieder nach unten, bekommt aber einen Schubs nach oben und flattert, wenn der Spieler auf den Bildschirm klickt. 
Wenn der Vogel durch ein Paar Rohre geflogen ist, bekommt der Spieler einen Punkt. 
Wenn der Vogel die Rohre oder die Unterkante berührt, ist das Spiel vorbei. 
Das Ziel des Spiels ist es, so viele Punkte wie möglich zu erzielen.

## Mach die Rohre

Zeichne eine Figur für die Rohre und sorge dafür, dass die Figur mehrere Kostümen hat, damit das Loch an verschiedenen Stellen ist.

<img src="image003.png" width="100%" />

Es ist möglich zu randomisieren, welche Kostüme verwendet wird.
Wenn du drei Kostümen hast, die Kostüme1, Kostüme2 und Kostüme3 heißen, benutze das <img src="image005.png" width="400" />, um eine zufällige Kostüme zu bekommen.

Füge den Code ein und teste ihn, damit sich die Röhren von rechts nach links bewegen. 
Ändere die Argumente, bis es gut aussieht.

<img src="image007.png" width="400" />

## Mach den Vogel

Wähle eine Figur mit zwei Kostümen oder zeichne deine eigene Figur.

Erstelle eine Variable <img src="y-Geschw.png" width="150" />, die angibt, wie viele Schritte sich der Vogel in y-Richtung bewegen soll, wenn der Computer tickt.
Die <img src="y-Geschw.png" width="150" /> Variable sollte mit jedem Tick abnehmen, damit der Vogel nach unten beschleunigt.

Wenn der Spieler irgendwo auf den Bildschirm klickt, sollte der y-Schritt einen Wert haben, der den Vogel nach oben springen lässt.

Füge Code hinzu und ändere die Parameter, bis es richtig funktioniert.

<img src="image011.png" width="400" />

## Spiel vorbei
Wenn der Vogel zu nah an der Unterkante ist oder die Rohre berührt, sollte er eine "Game over" Nachricht senden.

Füge den Code an der entsprechenden Stelle ein. Wo kann sie platziert werden?
<img src="image013.png" width="400" />

Denke daran, dass wenn du eine Figur ausblendest, du es einblenden musst, wenn das Programm startet.

Entscheide selbst, was bei Game Over passieren wird.

## Scoring
Erstelle eine Variable um die Punkte zu zählen und setze den Punktestand auf Null wenn das Programm startet.

Ziehe die Rohre so, dass sie sich links vom Vogel befinden.
Wenn ein Paar Rohre an dieser Position ist, sollte der Spieler einen Punkt bekommen, dann hat der Vogel es geschafft, an den Rohren vorbei zu kommen.
Schau dir die x-Koordinate der Rohre an und setze den Code an die entsprechende Stelle.

<img src="image015.png" width="400" />

Teste dein Programm und stelle sicher, dass es mäßig schwierig ist.
